## Возврат значения

### Через get

```
map.get(key)
```

Если ключ отсутствует - вернет undefined.

### Через has

```
map.has(key)
```

Возвращает boolean.

### Возврат текущего количества элементов

```
map.size 
```

## Add

```
map.set(key, value)
```

Каждый вызов map.set возвращает объект map, так что мы можем объединить вызовы в цепочку:

```
map.set("1", "str1")
  .set(1, "num1")
  .set(true, "bool1");
```

## Delete

### Удаление по ключу

```
map.delete(key)
```

### Полная очистка коллекции

```
map.clear()
```
